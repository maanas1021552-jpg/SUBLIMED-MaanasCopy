Prerequeisites:


CVODE 
    - Need to install using cmake3
    - Download and extract cvode package
    - cd in, and make directories sundials/builddir, sundials/srcdir, sundials/instdir
    - go into sundials/builddir and run > cmake3 ../../
    [- may be able to get away with just the builddir for a root install]
    - make
    - make install
CFITSIO
QHULL
GSL

Note - the packaged versions of CFITSIO, QHULL and GSL in this repository may be sufficient, avoiding the need for additional packages.

Set up the paths to the library and header files in Makefile.defs and Makefile
